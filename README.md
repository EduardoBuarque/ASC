ASC
===

ASC - Ajax Sales Cloud

Sistema completo para vendas de produtos em lojas fisicas, tendo como principal caracteristica a facilidade de ser executado via browser.



Tecnologia utilizada: 

	AJAX / PHP / MySQL
	
	
	
Sistema desenvolvido totalmente em Ajax com requisições a páginas PHP, contendo os seguintes recursos :

Cadastro
	- Clientes
	- Produtos
	- Coleções
	- Fornecedores
	
Vendas
	- Venda normal
	- Venda VIP 
	- Estorno de produto
	- Retornar produto VIP
	- Fechar turno
	
Financeiro
	- Fluxo de caixa
	
Relatórios
	- Produtos 
	- Vendas
	- Vendas VIP
	- Estoque
	- Fornecedores
	- Clientes
	
Utilitários
	- Agenda
	- Etiquetas
	- Mailling
	- Atualizações
	
Configurações
	- Backup
	- Administradores
	- Configurações
	
Ajuda
	- Manual
	- Ajuda on-line
